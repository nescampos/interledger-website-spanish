--- layout: developer-tools title: Herramientas testnet_wallets: - name: Rafiki Money description_html: >-

Una billetera de demostración orientada al usuario que puede realizar pagos de Interledger a un variedad de pares (compatible con USD, solo testnet). Regístrese en https://rafiki.money.

- name: RippleX Wallet description_html: >-

Una billetera de demostración orientada al usuario que puede realizar pagos de Interledger a un variedad de pares (compatible con XRP, solo testnet). Regístrese en https://wallet.ilpv4.dev.

connectors: - name: Conector Java description_html: >-

Un conector Interledger  implementado en Java. No se mantiene activamente, pero el equipo de desarrollo está abierto a escuchar sobre solicitudes de nuevas funciones.

- name: Conector Javascript description_html: >-

Un conector Interledger  implementado en Javascript. Este proyecto ha sido probado en batalla en varios despliegues de producción.

- name: Conector Rust description_html: >-

Un conector Interledger implementado en Rust. Este proyecto no se mantiene activamente, pero tiene un conjunto de características robusto y es fácil de usar.

libraries: - name: Interledger Java description_html: >-

Cree aplicaciones ILP que envíen y reciban pagos de forma nativa en Java utilizando Quilt, que es una implementación Java de Interledger.

- name: Interledger RS description_html: >-

Cree aplicaciones ILP que envíen y reciban pagos de forma nativa en Rust usando  interledger.rs, que es una implementación de Interledger en Rust.

- name: ILP-over-HTTP description_html: >-

Implementaciones de ILP- over-HTTP, que es un protocolo de comunicación bilateral para conexiones ILP de servidor a servidor.

  • Java: ILP-over-HTTP Link
  • Rust: ILP-over-HTTP
  • Javascript: ilp-plugin-http
- name: Interledger STREAM description_html: >-

Envíe dinero y datos en paquetes de manera confiable a través de Interledger usando STREAM.

  • Java: STREAM Java
  • Rust: STREAM RS
  • Javascript: STREAM JS
